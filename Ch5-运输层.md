# 2.用户数据报协议UDP

## 2.1UDP概述

1. UDP只在IP的数据报服务上增加了



主要特点

1. 无连接
2. 使用尽最大努力交付
3. 面向报文的
4. 没有拥塞控制
   + 因此网络出现的拥塞不会使源主机的发送率降低
5. 支持一对一、一对多、多对一和多对多的交互通信
6. 首部开销小









# 3.1TCP



## 3.1TCP特点

1. TCP是面向连接的运输层协议
2. 每一条TCP只能有两个端点
3. TCP提供可靠交付的服务
4. 提供全双工通信
5. 面向字节流
